DOCTYPE

  • dtd：document type definition 文档类型定义
  • doctype的作用就是告诉浏览器 dtd，让浏览器知道以什么规范渲染当前的文档

种类

  • html 5规范：!DOCTYPE html
  • html4.01 strict规范：!DOCTYPE HTML PUBLICE ... strict.dtd
    • 该dtd告诉浏览器可以渲染所有html元素和属性
    • 但是！不包括一些特定的元素，比如font
  • html4.01 transitional规范：!DOCTYEP ... loose.dtd
    • 该dtd告诉浏览器可以渲染所有html元素和属性
    • 包括一些展示性的和弃用的元素，比如font

浏览器的渲染过程

  • html结构 -> dom树
  • css规则 -> css style rules
  • dom树和css style rules结合生产 render树, 其中要进行layout，计算出dom节点的位置大小颜色等
  • 浏览器paint这个render，在浏览器上显示页面

reflow 回流

什么是reflow
dom结构中每个元素都有自己的盒子模型，浏览器结合各种style，通过计算layout，并根据计算结果将元素放到该出现的位置display，这个过程就是reflow
一次reflow，相当于是一次初始化过程
如何触发reflow
  • 增加、删除、修改dom，造成dom位置和大小改变的
  • 移动dom位置
  • 修改样式，比如：改变网页字体大小，改变元素宽高
  • resize窗口，scroll滚动的时候
  • 调用某个css的属性，比如：dom.clientHeight()
如何减少reflow
  • 对于动画，绝对定位，这样就只有repaint

repaint

什么是repaint
当各种盒子的大小位置颜色确定后，浏览器把他们绘制出来display到页面上
如何触发
  • 不该变dom的元素大小和位置
