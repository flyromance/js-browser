BFC原理

  • 垂直方向的间距发生重叠
  • bfc的区域不会与float元素重叠, 用于清除float
  • 独立的容器,不会影外面的元素
  • 计算高度的时候，float元素也参与其中

如何创建bfc

  • float: 非none，left right
  • position: 非static、relative，fixed、absolute
  • display: inline-block table table-cell
  • overflow: 非visible，hidden auto scroll

使用场景

  • bfc内，box边距重叠，外层套一个bfc
  • BFC的区域不会与float的元素区域重叠
  • 计算BFC的高度时，浮动子元素也参与计算
